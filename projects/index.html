Exploring A Digital Campus's Student Recruitment at an Organizational Level

2021
College of Engineering and Computing, George Mason University

Data Collection, Preprocessing, & Analysis, LDA Topic Modeling, Data Visualizations
Python, Tableau

A project where we collected data from Twitter using the Researcher API, and through Mason360, about GMU Student and Campus Organization's recruitment habits. We created a sorting system to work with the data, grouped similar tweets using LDA, and finally created recommendations that Organization's leaders can implement to better recruit online. The final deliverable for AIT 722, Theories and Models in Geo-Social Data Analytics.

Data Analysis (Collection, Preprocessing, & Visualization), LDA Topic Modeling
